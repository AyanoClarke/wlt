# wlt
USTC 网络通 fish 脚本

## Usage

```bash
fisher install AyanoClarke/wlt
wlt username passwd type exp
```

其中 type 是出口类型

| type | 描述 |
| -----|-----|
|  0 |教育网出口 |
|     1 | 电信出口 |
|     2 | 联通出口 |
|     3 | 电信出口2 |
|     4 | 联通出口2 |
|     5 | 电信出口3 |
|     6 | 联通出口3 |
|     7 | 教育网出口2 |
|     8 | 移动测试国际出口 |

 exp 是时间长度 

| exp | 描述 |
| ---- | ---- |
|   0     | 永久 |
|   120   | 动态 |
|   3600  | 1小时|
|   14400 | 4小时|
|  39600 | 11小时 |
|  50400 | 14小时 |
